## Enhancements

### Redis Cloud Essentials support for Terraform

The [Redis Cloud Terraform provider]() now supports provisioning Redis Cloud Essentials databases. It also supports Active-Active databases with advanced capabilities. 

See the [Redis Cloud Terraform provider changelog](https://github.com/RedisLabs/terraform-provider-rediscloud/blob/main/CHANGELOG.md) for full version content.

See [Get started with Terraform]() to learn how to install the Redis Cloud provider and create a subscription and database.
