This changelog lists new features, enhancements, and other changes added to Redis Cloud during May 2023.

## New features and enhancements

### Cloud API - Fixed plans support

The [Redis Cloud API]() supports new endpoints to manage fixed subscriptions and databases. 

The new fixed endpoints can be viewed in the [Redis Cloud Swagger UI](https://api.redislabs.com/v1/swagger-ui/index.html#/Subscriptions%20-%20Fixed). For more information about using the Redis Cloud API, see [Use the REST API]().

### Active-Active access roles

For [Active-Active databases](), you can now define roles with different levels of access for different regions.

For more information, see [Active-Active roles]().
