## Run Redis Open Source on Windows using Docker

To install Redis Open Source on Windows, you will need to have Docker installed. When Docker is up and running, open Windows PowerShell and follow the instructions described in [Run Redis Open Source on Docker](). Then, use Docker to connect with `redis-cli` as explained in that topic.
