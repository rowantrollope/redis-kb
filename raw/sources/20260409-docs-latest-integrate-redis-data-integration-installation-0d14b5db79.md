The guides in this section explain the options you have for installing and upgrading RDI on your own servers. See the [Redis Cloud RDI guide]() to 
learn how to set up RDI for a cloud database.
Before you use RDI, you must also configure your source database to enable CDC. See the
[Prepare source databases]()
section to learn how to do this.
